=begin
---Instruções---

Botão para Correr: SHIFT
Modifique a velocidade nas linhas: 17 e 19

Se quiser mudar o botão de correr mude no (Input::SHIFT) para outra
tecla.
=end

class Game_Character
  attr_accessor :move_speed
end
class Scene_Map
  alias new_update update
  def update
    new_update
    if Input.press?(Input::SHIFT)
      $game_player.move_speed = 3
    else
      $game_player.move_speed = 2
  end
  end
end
